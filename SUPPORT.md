# Support & FAQ - 3things

**Last Updated:** January 11, 2026

## 📚 快速开始 (Quick Start)

### 安装方法

#### 从 Chrome Web Store 安装 (推荐)
1. 访问 Chrome Web Store (即将上线)
2. 点击"添加到 Chrome"
3. 确认权限
4. 开始使用！

#### 手动安装 (开发版)
1. 下载最新版本的 ZIP 文件
2. 解压到任意目录
3. 打开 Chrome 浏览器
4. 访问 `chrome://extensions/`
5. 启用右上角的"开发者模式"
6. 点击"加载已解压的扩展程序"
7. 选择解压后的文件夹

### 基本使用

**添加任务:**
- 打开新标签页
- 在输入框中输入任务
- 按回车或点击"+"按钮
- 最多可以添加 3 个任务

**完成任务:**
- 点击任务左侧的圆圈
- 圆圈会变为实心，任务文字会出现删除线
- 再次点击可取消完成状态

**删除任务:**
- 将鼠标悬停在任务上
- 点击右侧出现的垃圾桶图标

**查看历史:**
- 点击右上角的历史记录图标
- 查看过往所有任务记录

**快速访问:**
- 点击浏览器工具栏的扩展图标
- 在弹窗中快速查看当前任务

## ❓ 常见问题 (FAQ)

### Q1: 为什么限制只能添加 3 个任务？

**A:** 这是 3things 的核心理念。通过限制为 3 个任务，强迫你进行优先级排序，专注于真正重要的事情。研究表明，每天完成 3 件重要的事情比列出 10 件但只完成 2 件更有成就感。

### Q2: 我的数据会丢失吗？

**A:** 你的数据存储在浏览器的本地存储中。一般情况下不会丢失，但以下情况可能导致数据丢失：
- 清除浏览器数据
- 卸载扩展
- 更换电脑或浏览器

建议定期导出你的数据（未来版本会加入导出功能）。

### Q3: 数据是否同步到云端？

**A:** 目前不会。所有数据都只存储在你的本地浏览器中，这是为了保护你的隐私。我们不会访问、收集或传输你的任何数据。

### Q4: 如何备份我的任务数据？

**A:** 目前版本可以通过以下方式手动备份：
1. 点击历史记录图标
2. 截图保存重要记录

我们正在开发 CSV 导出功能，将在未来版本中提供。

### Q5: 扩展会记录我的浏览历史吗？

**A:** 完全不会。3things 只请求了 `storage` 权限，用于在本地保存你的任务数据。我们不追踪、不记录你的任何浏览行为。

### Q6: 为什么新标签页显示的是英文？

**A:** 3things 目前主要支持中文界面。如果看到英文，可能是浏览器语言设置的问题。我们正在添加多语言支持，包括英文。

### Q7: 如何更改字体或主题？

**A:** 目前版本不支持自定义主题和字体。我们设计了精心挑选的配色和字体，以提供最佳的用户体验。如果你有强烈的需求，可以在 GitHub 上提 feature request。

### Q8: 扩展会拖慢我的浏览器吗？

**A:** 不会。3things 非常轻量级：
- 包大小 < 200KB
- 只在新标签页加载
- 无后台进程
- 无网络请求
- 无第三方库

### Q9: 如何卸载扩展？

**A:**
1. 打开 `chrome://extensions/`
2. 找到 3things
3. 点击"移除"
4. 确认删除

**注意:** 卸载会删除所有本地存储的任务数据，请提前截图保存重要内容。

### Q10: 支持其他浏览器吗？

**A:** 3things 基于 Chrome 扩展标准构建，理论上支持所有基于 Chromium 的浏览器：
- Microsoft Edge
- Brave
- Opera
- Vivaldi

但我们会优先在 Chrome 上测试和优化。

### Q11: 如何更新到最新版本？

**A:**
- **Chrome Web Store 版本**: 浏览器会自动更新
- **手动安装版本**: 需要重新下载和安装

更新前建议截图保存当前任务。

## 🔧 技术支持

### 系统要求

- **浏览器:** Chrome 88+ 或其他基于 Chromium 的浏览器
- **权限:** storage (本地存储)
- **网络:** 不需要网络连接
- **存储空间:** < 1MB

### 权限说明

**storage 权限:**
- **用途:** 保存你的任务数据到浏览器本地存储
- **数据内容:** 仅包含你的任务文本和完成状态
- **数据位置:** 仅存储在你的设备上
- **数据传输:** 从不传输到任何服务器

### 故障排除

#### 扩展无法加载
**解决方案:**
1. 确认浏览器版本 >= 88
2. 尝试重新加载扩展
3. 检查浏览器控制台是否有错误信息
4. 重新安装扩展

#### 任务不保存
**可能原因:**
- 浏览器隐私模式（Incognito Mode）
- 浏览器设置禁止存储

**解决方案:**
- 在正常模式下使用
- 检查浏览器存储设置

#### 界面显示异常
**解决方案:**
1. 清除浏览器缓存
2. 禁用其他可能冲突的扩展
3. 检查浏览器缩放比例（建议 100%）
4. 更新到最新版本

#### 历史记录不更新
**解决方案:**
1. 刷新页面
2. 检查是否已添加新任务
3. 重新打开历史模态框

## 🔐 隐私与安全

### 隐私承诺

3things 承诺：
- ❌ 不收集任何个人数据
- ❌ 不使用任何追踪工具
- ❌ 不连接任何远程服务器
- ❌ 不包含任何第三方广告
- ❌ 不出售或分享数据

### 数据安全

你的数据存储方式：
- 使用 Chrome Storage API
- 仅存储在本地浏览器
- 使用标准的 localStorage 加密（浏览器自带）
- 不传输任何数据

### 安全建议

- 定期截图保存重要任务
- 不要在公共电脑上使用（或使用后清理）
- 卸载前备份数据
- 保持浏览器和扩展更新
