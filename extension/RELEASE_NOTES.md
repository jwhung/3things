## 📦 安装方式

### 方式一：Chrome Web Store（推荐）

> 即将上线，敬请期待...

### 方式二：手动安装

1. 下载 [3things-extension-v1.0.1.zip](https://github.com/jwhung/3things/releases/download/v1.0.1/3things-extension-v1.0.1.zip)
2. 解压 ZIP 文件
3. 打开 Chrome 浏览器，访问 `chrome://extensions/`
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压后的文件夹
7. 完成！🎊
